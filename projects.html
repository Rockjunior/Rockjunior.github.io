IRS Project



This project evaluates data-handling skills using an IRS dataset. Candidates analyze and clean the dataset, identifying untidy data issues, resolving duplicates, and filling missing values. Tasks include summarizing data by county and sub-county, such as spray coverage, net usage, and reasons for unsprayed rooms, with visual representations for clarity. Candidates also provide insights on record discrepancies per data collector (SOP Code) and outline quality assurance measures for data collection. The project requires candidates to work in Excel, applying data cleaning, summarization, and visualization techniques, with all work completed within a strict 2-hour timeframe.
